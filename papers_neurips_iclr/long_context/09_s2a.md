**Authors:** Jason Weston, Sainbayar Sukhbaatar (Meta AI / FAIR)

**Venue:** arXiv preprint (cs.CL, cs.AI, cs.LG), 2023

**arXiv:** [https://arxiv.org/abs/2311.11829](https://arxiv.org/abs/2311.11829)

## Abstract

Soft attention in Transformer-based Large Language Models can be compromised by irrelevant or biasing context information, leading to degraded response quality. The authors propose System 2 Attention (S2A), a technique that leverages the reasoning and instruction-following abilities of LLMs to decide what to attend to. S2A regenerates the input context to contain only the portions relevant to the query, then attends to this regenerated context to produce the final response. Experiments show improvements on tasks involving question answering with irrelevant context, math word problems with opinions, and long-form text generation, with gains in factuality, objectivity, and reduced sycophancy.

---

## 1. Introduction

### 1.1 The Problem with Soft Attention

Standard Transformer attention (System 1, in the Kahneman framework) assigns weights to all input tokens based on learned patterns. This can lead to:
- Attending to irrelevant or misleading context
- Sycophantic behavior (agreeing with user opinions embedded in context)
- Distraction by opinionated or biased framing in prompts

### 1.2 System 1 vs. System 2

Drawing from cognitive science (Kahneman's dual-process theory):
- **System 1:** Fast, automatic, pattern-based -- analogous to standard soft attention
- **System 2:** Slow, deliberate, logical -- the proposed S2A mechanism

S2A introduces a deliberate reasoning step before the final generation, filtering context through the LLM's own reasoning capabilities.

## 2. Method: System 2 Attention

### 2.1 Two-Step Process

**Step 1: Context Regeneration**
The LLM is prompted to rewrite the input, removing irrelevant or potentially biasing information. The instruction asks the model to extract only the facts and context relevant to answering the question.

**Step 2: Response Generation**
The LLM generates its final response using only the filtered/regenerated context from Step 1.

### 2.2 Prompt Design

The S2A prompt instructs the model:
> "Given the following text by a user, extract the part that is unbiased and not their opinion, so that using that text alone would be good context for providing an unbiased answer to the question portion of the text."

This leverages the LLM's instruction-following capability to perform deliberate context filtering.

### 2.3 Variants

- **S2A (Instruct):** Uses instruction-tuned models for both steps
- **S2A (Few-shot):** Provides examples of context filtering in the prompt
- **S2A + Contrastive:** Combines with contrastive decoding for additional debiasing

## 3. Experimental Evaluation

### 3.1 Factual QA with Irrelevant Context

Tasks where questions are paired with opinions or irrelevant information:
- S2A successfully strips opinionated framing
- Reduces sycophancy (tendency to agree with stated opinions)
- Improves factual accuracy on knowledge-based questions

### 3.2 Math Word Problems

GSM-IC benchmark (math problems with irrelevant context sentences):
- Standard attention is distracted by irrelevant numerical information
- S2A filters out irrelevant sentences before solving
- Significant accuracy improvement on problems with distractors

### 3.3 Long-Form Text Generation

Evaluated on biography and opinion-based generation tasks:
- **Factuality:** S2A produces more factually accurate text
- **Objectivity:** Reduced influence of user opinions on generated content
- **Sycophancy reduction:** Model less likely to parrot back user's stated preferences

### 3.4 Quantitative Results

| Task | Standard | S2A | Improvement |
|---|---|---|---|
| Sycophancy (factual QA) | High sycophancy rate | Significantly reduced | Large |
| GSM-IC (math + distractors) | Accuracy drops with distractors | Maintained accuracy | Moderate |
| Long-form objectivity | Biased by context | More objective | Moderate |

## 4. Analysis

### 4.1 What Gets Filtered

Analysis of the regenerated contexts shows S2A effectively removes:
- User opinions and preferences
- Irrelevant numerical information in math problems
- Leading/suggestive framing
- Redundant or distracting context

### 4.2 Computational Cost

S2A requires two LLM forward passes instead of one, approximately doubling inference cost. The authors argue this is acceptable for applications where accuracy and objectivity are critical.

### 4.3 Failure Modes

- Occasionally removes relevant context if it appears opinion-like
- Less effective when bias is subtle or deeply embedded in factual statements
- Depends on the LLM's ability to distinguish relevant from irrelevant content

## 5. Related Work

- **Attention mechanisms:** Standard soft attention, sparse attention, hard attention
- **Debiasing:** Contrastive decoding, calibration methods
- **Chain-of-thought:** Related idea of intermediate reasoning before answering
- **Context distillation:** Compressing context to essential information

S2A differs by using the LLM itself as the filter, requiring no additional training or architectural changes.

## 6. Conclusion

System 2 Attention demonstrates that LLMs can improve their own attention by using deliberate reasoning to filter context before generation. The approach is simple, requires no training, and shows consistent improvements on tasks where irrelevant or biasing context degrades performance. The analogy to dual-process theory in cognitive science provides a useful framework for understanding and improving LLM attention mechanisms.
